EMERGENCY — Always Treat First

Anaphylaxis
➡ IM Epinephrine FIRST
  • Then: Oxygen + IV fluids
  • Then: Antihistamines + Steroids
  • Never delay epinephrine
Acute Angle-Closure Glaucoma
  • Severe eye pain + halos
  • Headache + nausea
  • Fixed mid-dilated pupil
➡ IV Acetazolamide + Ophthalmology
Testicular Torsion
  • Sudden severe testicular pain
  • High-riding testicle
  • Absent cremaster reflex
  • 6-hour window for salvage
➡ Immediate surgical exploration
Epidural Hematoma
  • Trauma → lucid interval → rapid decline
  • Middle meningeal artery
➡ CT: Biconvex (lens-shaped)
Subdural Hematoma
  • Elderly / alcoholic / minor trauma
  • Bridging veins
➡ CT: Crescent-shaped
Epiglottitis
  • Drooling + stridor + tripod position
  • Toxic-looking child
  • Do NOT examine throat
➡ Secure airway FIRST
❤️

Cardiology

Stable Angina
  • Chest pain with exertion
  • Relieved by rest / nitroglycerin (<5 min)
➡ Stress test for Dx
Acute MI (STEMI)
  • Crushing chest pain → arm/jaw
  • Diaphoresis + nausea
➡ ECG first → PCI <90 min
  • ASA + P2Y12 + anticoagulation (MONA)
Aortic Dissection
  • Tearing chest pain → back
  • BP difference between arms
  • Widened mediastinum on CXR
➡ CT Angiography
  • Type A → surgery | Type B → labetalol
Atrial Fibrillation
  • Irregularly irregular pulse
  • No discrete P waves on ECG
➡ CHA₂DS₂-VASc → DOAC
  • Rate control: beta-blocker / CCB
Aortic Stenosis
  • Classic triad: Syncope + Angina + Dyspnea
  • Harsh systolic murmur → carotids
➡ Echo → AVR when symptomatic
Heart Failure
  • Dyspnea + orthopnea + bilateral edema
  • Bilateral crackles
➡ BNP + Echo
  • HFrEF: ACEi + BB + diuretic + spironolactone
🫁

Respiratory

Pulmonary Embolism
  • Dyspnea + pleuritic chest pain + tachycardia
  • Normal CXR does NOT rule out PE
➡ CT Pulmonary Angiography (high prob)
  • D-dimer for LOW probability only
Pneumothorax
  • Absent breath sounds + hyperresonance
  • Tension: trachea deviates AWAY + hypotension
➡ Tension: Needle decompression 2nd ICS MCL
  • Simple large: chest tube
Asthma
  • Episodic wheeze + cough + chest tightness
  • Worse at night + reversible
➡ SABA PRN + ICS maintenance
COPD Exacerbation
  • ↑ Dyspnea + ↑ sputum + purulence
  • Smoker + chronic disease
➡ SABA + prednisone 40mg + antibiotics
  • O₂ target: 88–92%
Tuberculosis
  • Chronic cough + night sweats + weight loss
  • Upper lobe cavitation on CXR
➡ Sputum AFB + IGRA
  • RIPE x2mo → RI x4mo; ISOLATE
Community-Acquired Pneumonia
  • Fever + productive cough + lobar consolidation
  • MCC: S. pneumoniae
➡ CURB-65 guides admission
  • Outpatient: amoxicillin or azithromycin
🫀

Gastroenterology

Acute Pancreatitis
  • Epigastric pain radiating to back
  • Lipase >3x ULN
  • MCC: Gallstones + Alcohol
➡ IV fluids (LR) + NPO + analgesia
Appendicitis
  • RLQ pain + fever + nausea
  • McBurney point tenderness
➡ CT abdomen → laparoscopic appendectomy
  • Always β-hCG in females
Peptic Ulcer Disease
  • Duodenal — pain relieved by food + nocturnal
  • Gastric — pain worse with food + weight loss
  • H. pylori + NSAIDs = MCC
➡ PPI + H. pylori eradication
Upper GI Bleed
  • Melena = black tarry stool
  • MCC: PUD → varices
➡ IV PPI + endoscopy within 24hr
Ulcerative Colitis vs Crohn
  • UC — bloody diarrhea + rectum always + continuous
  • Crohn — any GI segment + skip lesions + fistulas + perianal
➡ Colonoscopy + biopsy
Mesenteric Ischemia
  • Pain OUT OF PROPORTION to exam
  • A-fib → embolic MCC
➡ CT angiography → surgical emergency
🧠

Neurology

Acute Stroke
  • Sudden focal weakness / speech / vision
  • Ischemic 85% vs Hemorrhagic 15%
➡ CT head (no contrast) FIRST
  • tPA if ischemic + within 4.5hr
Subarachnoid Hemorrhage
  • "Worst headache of life" — thunderclap
  • Ruptured berry aneurysm
➡ CT head → if negative → LP (xanthochromia)
  • Nimodipine for vasospasm
Parkinson Disease
  • TRAP: Tremor (resting pill-rolling) + Rigidity + Akinesia + Postural instability
  • Micrographia + shuffling gait
➡ Levodopa/Carbidopa (most effective)
Guillain-Barré Syndrome
  • Ascending flaccid paralysis + areflexia
  • Post-infectious (Campylobacter)
  • CSF: ↑protein + normal WBC
➡ IVIG or Plasmapheresis
Bell's Palsy vs Stroke
  • Bell's — forehead INVOLVED (LMN)
  • Stroke — forehead SPARED (UMN)
➡ Prednisone + Valacyclovir within 72hr
Meningitis
  • Headache + photophobia + neck stiffness
➡ Antibiotics FIRST if delayed LP risk
  • Ceftriaxone + vancomycin ± ampicillin + dexamethasone
⚗️

Endocrinology

Hyperthyroidism
  • Weight loss + heat intolerance + tremor + palpitations
  • Low TSH + high free T4
➡ Methimazole → radioiodine or surgery
Hypothyroidism
  • Weight gain + cold intolerance + fatigue + constipation
  • High TSH + low free T4
➡ Levothyroxine (synthetic T4)
Cushing Syndrome
  • Moon face + central obesity + purple striae
  • Buffalo hump + hypertension
  • MCC: Exogenous steroids
➡ 24hr urine cortisol + overnight dexamethasone suppression
Addison Disease
  • Hypotension + hyperpigmentation
  • Hyponatremia + hyperkalemia + hypoglycemia
➡ IV hydrocortisone + saline (crisis)
Pheochromocytoma
  • Episodic HTN + headache + sweating + palpitations
➡ Plasma free metanephrines
  • Alpha-blocker FIRST → then beta-blocker → surgery
DKA vs HHS
  • DKA — ketones + acidosis + type 1
  • HHS — glucose >33 + NO ketones + type 2
➡ IV fluids + insulin + electrolytes
🫘

Renal

Kidney Stones (Renal Colic)
  • Severe colicky flank pain → groin
  • Hematuria + nausea
  • Patient writhes (cannot stay still)
➡ CT KUB (no contrast) + NSAIDs
  • Tamsulosin for stones <10mm
Nephrotic Syndrome
  • Proteinuria >3.5g/day + edema
  • Hypoalbuminemia + hyperlipidemia
➡ Renal biopsy for Dx
  • MCC adults: FSGS | children: minimal change
UTI Spectrum
  • Cystitis — dysuria + frequency; no fever
  • Pyelonephritis — fever + CVA tenderness
➡ Nitrofurantoin (cystitis) | Ceftriaxone (pyelo)
Hyperkalemia (Dangerous)
  • Peaked T waves → wide QRS → VF
  • ACE inhibitors + renal failure
➡ IV Calcium gluconate FIRST (stabilize membrane)
👶

Pediatrics

Croup
  • Barking/seal cough + inspiratory stridor
  • Worse at night | Age 6mo–3yr
  • Parainfluenza virus
➡ Dexamethasone 0.6mg/kg PO + neb epinephrine (severe)
Pyloric Stenosis
  • Projectile NON-bilious vomiting at 2–8 weeks
  • Hypochloremic metabolic alkalosis
  • Olive-shaped mass
➡ Ultrasound → correct electrolytes → Pyloromyotomy
Intussusception
  • Currant jelly stool + episodic crampy pain
  • Age 6mo–2yr | Sausage-shaped mass
➡ US (target sign) → Air enema reduction
Kawasaki Disease
  • Fever >5d + conjunctivitis + strawberry tongue + rash + cervical LAD + extremity changes
➡ IVIG + high-dose ASA + Echo (coronary aneurysms)
Bronchiolitis
  • Infant + URI + wheeze + crackles
  • MCC: RSV
➡ Supportive ONLY (no steroids/bronchodilators)
Neonatal Jaundice Timing
  • <24hr — PATHOLOGIC (hemolysis)
  • Day 2–7 — Physiologic
  • Day 5–14 — Breast milk
➡ <24hr = always investigate urgently
🧩

Psychiatry

Depression
  • Low mood + anhedonia ≥2 weeks
  • SIGECAPS criteria
➡ SSRI first-line + CBT
Bipolar Disorder
  • Mania: grandiosity + ↓sleep + risky behavior
➡ Lithium (mood stabilizer)
  • Monitor: Li level + TSH + renal function
  • NSAIDs → lithium toxicity
Panic Disorder
  • Recurrent unexpected panic attacks
  • Anticipatory anxiety
➡ SSRI/SNRI + CBT
  • Benzodiazepines PRN short-term only
Schizophrenia
  • Delusions + hallucinations + disorganization
  • Symptoms >6 months
➡ Antipsychotics (olanzapine, risperidone)
NMS vs Serotonin Syndrome
  • NMS — rigidity + fever + antipsychotics → dantrolene
  • SS — clonus + hyperreflexia + SSRIs → cyproheptadine
Delirium vs Dementia
  • Delirium — ACUTE + fluctuating + reversible
  • Dementia — chronic + progressive
➡ Delirium: find and treat the CAUSE first

Clinical Criteria & Scores

▼

🫀 CHA₂DS₂-VASc — Atrial Fibrillation

  • CHF (1) | HTN (1) | Age >75 (2) | DM (1)
  • Stroke/TIA/TE (2) | Vascular disease (1) | Age 65-74 (1) | Female (1)
  • Max = 9 (can't count age twice)
Score → Tx:
≥2 → Warfarin/Xa blocker/DTI  |  <2 → Aspirin

🦠 Centor Score — Strep Pharyngitis

  • Cough (NONE) | Exudates | Anterior cervical LN | Temp >38
  • Age 3-14 (+1) | Age 15-44 (0) | Age 44+ (-1)
Score → Action:
4+ → Penicillin  |  2-3 → Rapid strep test  |  ⚠️ Children: always rapid strep

🫁 CURB-65 — Pneumonia Severity

  • Confusion | Urea (BUN >7) | RR >30 | BP low (SBP<90/DBP<60) | Age ≥65
>2 → Hospitalize
Inpatient: Fluoroquinolone  |  Outpatient Atypical: Azithromycin  |  Outpatient Typical: Amoxicillin

🩸 SEPSIS Criteria

SIRS = 2+ of: Temp >38/<36 | HR>90 | RR>20 or PaCO₂<32 | WBC>12k/<4k/>10% bands
  • Sepsis = SIRS + Source of infection
  • Severe Sepsis = SIRS + Source + End organ damage + HoTN + Lactate >4
  • Septic Shock = Severe sepsis unresponsive to fluids

💧 Light's Criteria — Pleural Effusion

  • Pleural protein/Serum protein >0.5 = Exudate
  • Pleural LDH/Serum LDH >0.6 = Exudate
  • Pleural LDH >2/3 upper normal = Exudate

🫁 Well's Criteria — PE

Score:
>4 → CTA  |  <4 → D-dimer
⚠️ If suspect PE → Give HEPARIN before CTA!

🏥 ACS Treatment — MONACBASH

MONACBASH:

⚡ Arrhythmias

  • SVT (narrow QRS): Stable→Vagal→Adenosine | Unstable→Cardioversion
  • VTach (wide QRS): Stable→Amiodarone | Unstable→Cardioversion
  • V-fib / Pulseless VTach → Defibrillation
  • Asystole / PEA → CPR
  • Torsades de Pointes → IV Magnesium

🫀 AV Blocks

  • 1st degree → No tx
  • Mobitz I (Wenckebach) → No tx
  • Mobitz II → Pacemaker
  • Complete AV Block → Pacemaker

🫀 CHF

Decrease mortality: ACEi + BB + Spironolactone
Acute Decompensation — NO LIP:
Nitrates | Oxygen | Loop diuretics | Inotropes | Positioning

HTN Emergency vs Urgency

  • Emergency: 180/120 + end organ damage → IV hydralazine / Nitroprusside / Labetalol
  • Urgency: 180/120 + no end organ damage → Oral medications

🫀 Cardiac Tamponade

Beck's Triad:
Hypotension | JVD | Muffled heart sounds
  • Pulsus paradoxus: inspiration → >10 drop in SBP
  • Electrical alternans on EKG

  • Type B (distal to L subclavian) → Beta-blocker
  • Dx: CT angio (if no CKD) or TE Echo

Pulmonary

▼

COPD Stages (FEV1)

  • Mild >80% → Albuterol
  • Moderate 50-80% → Albuterol + Ipratropium
  • Severe 30-50% → Albuterol + Ipratropium + Inhaled steroid
  • Very Severe <30% → All above + Long-term O₂
Home O₂ indications: PaO₂ <55 OR O₂ sat <88%
Decrease mortality: Smoking cessation + Home oxygen

Asthma Stages

  • Intermittent (<2x/week) → Albuterol PRN
  • Mild persistent (3-7x/week) → Albuterol + Low dose ICS
  • Moderate persistent (daily) → Albuterol + Medium dose ICS
  • Severe persistent (multiple/day) → Albuterol + High dose ICS
⚠️ Normal/elevated PCO₂ in asthma attack → INTUBATE!

Mechanical Ventilation

Renal

▼

🚨 Emergent Hemodialysis — AEIOU

AEIOU:
Acidosis | Electrolytes (K >6.5) | Intoxication (MALE: Methanol/Aspirin/Lithium/Ethylene glycol) | Overload (fluid) | Uremia

🔬 Urine Casts

  • Muddy brown casts → ATN (shock/toxins)
  • WBC casts → AIN or Pyelonephritis
  • AIN = FEAR: Fever | Eosinophilia | Azotemia | Rash

🧪 RTA Types

  • RTA 1 → Can't secrete H⁺ → kidney stONEs
  • RTA 2 → Can't absorb bicarb → acidotic
  • RTA 4 → HYPOaldosterone → HYPERkalemia (only one with ↑K⁺)

💊 Hyponatremia Algorithm

  • Check osmolality: >295=hyper | 275-295=iso | <275=hypo
  • Check volume: Hyper | Eu | Hypo-volemic
  • Euvolemic + high urine Na → SIADH (Tx: fluid restrict)
  • Hypovolemic + symptoms → 3% hypertonic saline (correct slowly!)
⚠️ Overcorrect too fast → Central Pontine Myelinolysis!

Hematology / Oncology

▼

Microcytic Anemia — FAST

FAST:
Fe deficiency (↓ferritin, ↓serum Fe, ↑TIBC) | Anemia of chronic disease (↑ferritin, ↓serum Fe, ↓TIBC) | Sideroblastic (↑ferritin, ↑serum Fe, ↓TIBC) | Thalassemia

TTP vs HUS

  • TTP = FAT RN: Fever | Anemia | Thrombocytopenia | Renal | Neuro → Tx: Plasma exchange
  • HUS = EHEC/Shigella/Uremia → Hematuria + anemia + thrombocytopenia → Tx: Dialysis
⚠️ NEVER give abx for EHEC → predisposes to HUS!

🩸 HIT (Heparin-Induced Thrombocytopenia)

  • Plts drop ~50% within 3 days of heparin
  • STOP heparin → start argatroban/dabigatran (DTI)
  • Test: +PF4 Ab + serotonin release assay

DIC

  • ↑PT + ↑PTT + ↑BT + ↑D-dimer + ↓fibrinogen + thrombocytopenia
  • Tx: Treat underlying cause + FFP + Vitamin K

Lupus Criteria — SOAPBRAINMD (4/11)

SOAPBRAINMD:
Serositis | Oral ulcers | Arthralgias | Photosensitivity | Blood (pancytopenia) | Renal | ANA | Immunoglobulin (anti-dsDNA) | Neuro | Malar rash | Discoid lupus

🦶 Gout vs Pseudogout

  • Gout: Negatively birefringent crystals (yellow, needle-shaped) → MTP joint
  • Pseudogout (CPPD): Positively birefringent (blue, rhomboid) → chondrocalcinosis
  • Next step: Arthrocentesis
  • Tx: NSAIDs + Colchicine (CI: kidney disease)

Polymyalgia vs Polymyositis vs Fibromyalgia

  • Polymyalgia Rheumatica = STIFFNESS → elderly, ↑ESR, normal CPK → low dose steroids
  • Polymyositis = WEAKNESS → ↑CPK → steroids. Assoc with malignancy!
  • Fibromyalgia = PAIN → trigger points → TCA/SNRI
⚠️ Giant cell arteritis → HIGH DOSE steroids (prevents blindness!)

Fluids & Electrolytes

▼

⚡ Hyperkalemia Treatment

  1. Check EKG → if changes → Calcium gluconate (stabilize membranes)
  2. Insulin + Dextrose (shift K into cells)
  3. Kayexalate (excrete K via GI)

🦴 Calcium Disorders

  • Hypercalcemia → Tx: IVF
  • Hypermagnesemia: 1st sign = loss of DTRs → Tx: IV calcium gluconate
  • Blood transfusions → hypocalcemia (citrate binds Ca)

🧪 PTH Axis (High Yield!)

  • PTH → ↑Ca²⁺ + ↓phosphate (bone + kidney + vit D activation)
  • Primary hyperPTH → ↑PTH, ↑Ca, ↓P → Tx: parathyroidectomy
  • Kidney failure → ↑PTH with ↓Ca + ↑P (can't activate vit D)
  • Vit D deficiency → ↓Ca, ↓P, ↑PTH
  • SqCC lung → PTHrP → ↑Ca, ↓P, ↓PTH
